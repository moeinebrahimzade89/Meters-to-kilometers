# 📏 Meter to Kilometer Converter

A lightweight Python script that converts a distance value from **meters** to **kilometers**. It includes robust error handling to manage non-numeric inputs gracefully.

## 🛠 Technologies Used
- Python 3.x

## ✨ Features
- **Accurate Conversion:** Easily converts meters to kilometers.
- **Robust Error Handling:** Uses `try-except` blocks to catch invalid inputs (e.g., if a user types a string instead of a number).
- **User-Friendly:** Simple and clear interface for quick calculations.

## 🚀 How to Run
1. Save the code as `converter.py`.
2. Open your terminal or command prompt.
3. Run the following command:
